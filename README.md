# SQL_Employee_Tracker

By David Warner

## Description

CLI Inquirer app that navigates, creates, and update entries in a SQL database.

## Usage

1. Clone the repo
2. Seed the data through MySQL (if you want to utilize the provided seed data)
3. node index.js
